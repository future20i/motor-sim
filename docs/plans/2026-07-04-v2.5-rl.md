## Task 2: 创建 RL 环境 `rl_environment.py`

路径: `/root/workspace/motor-sim/rl_environment.py`

要点:
- 继承 `gymnasium.Env`
- 状态空间: [f, df, Vdc, dVdc, SOC, P_load, omega, temp] 8维连续
- 动作空间: Box(3,) → [mode_select, droop, dc_mode_select] 离散+连续混合
- 每个 step 推进一个控制周期 (50μs × 100 = 5ms)
- 奖励 = vdc_score + freq_score - soc_penalty - fault_penalty
- 使用 system_config DEFAULT_CONFIG

## Task 3: 创建训练脚本 `train_rl.py`

路径: `/root/workspace/motor-sim/train_rl.py`

要点:
- 使用 PPO 算法
- 训练 500,000 timesteps
- 每 50k 步保存 checkpoint
- 训练结束后导出 ONNX 模型

## Task 4: 验证训练结果

- 加载训练好的模型
- 跑 10 个 S4 场景对比评分
- 对比当前启发式规则树的得分

## Task 5: 创建决策日志 `rl_decision_logger.py`

路径: `/root/workspace/motor-sim/rl_decision_logger.py`

要点:
- 记录每次决策: t, obs, action, reason, confidence
